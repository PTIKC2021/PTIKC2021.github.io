BACKGROUND

PTIK C 2021 is one of the classes in the informatics and computer engineering department of the informatics and computer engineering education study program.

The association of students who are members of the PTIK C class in the class of 2021. The class which was formed on August 15, 2021 totals 42 students and comes from various regions living in Indonesia.

Like classes in general, of course, the first glance of each other still maintains an attitude. Over time, we bonded and created a cohesive classroom atmosphere.

BOOTSTRAP

framework, Februari 15, 2022

Bootstrap adalah kerangka kerja CSS yang sumber terbuka dan bebas untuk merancang situs web dan aplikasi web. Kerangka kerja ini berisi templat desain berbasis HTML dan CSS untuk tipografi, formulir, tombol, navigasi, dan komponen antarmuka lainnya, serta juga ekstensi opsional JavaScript. Tidak seperti kebanyakan kerangka kerja web lainnya, kerangka kerja ini hanya fokus pada pengembangan front-end saja.

W3SCHOOLS

Web Developer, Februari 18, 2022

W3Schools adalah situs web developer informasi, dengan tutorial dan referensi yang berkaitan dengan topik pengembangan web seperti HTML, CSS, JavaScript, PHP, SQL, dan JQuery. Situs ini mendapatkan namanya dari singkatan untuk World Wide Web; W3 adalah numeronym dari WWW. W3Schools tidak berafiliasi dengan W3C. Hal ini dibuat dan dimiliki oleh Refsnes data, pengembangan perangkat lunak milik keluarga Norwegia dan perusahaan konsultan. Situs ini menyediakan panduan referensi yang mencakup banyak aspek pemrograman web, termasuk teknologi seperti HTML, XHTML, CSS, XML, JavaScript, PHP, ASP, SQL dll.
